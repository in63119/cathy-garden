# Cathy Garden Task Plan

## 목적

이 문서는 Cathy Garden 프로젝트를 기존 구조에서 개인용 사진/영상 보관함 서비스로 전환하기 위한 전체 작업 목록을 정리합니다.

기준 방향은 다음과 같습니다.

- `Next.js + TypeScript + App Router`
- `Vercel` 배포
- `S3` 또는 `R2` 기반 오브젝트 스토리지
- 아내 1인을 위한 private archive
- 모바일 우선 경험
- 모든 코드 변경 후 테스트 수행
